# airsim_controller_ros

## setup
```
git clone git@github.com:microsoft/AirSim.git  
cd AirSim  
./setup.sh  
./build.sh 
./install_unreal.sh
cd UnrealEngine
./Setup.sh  
./GenerateProjectFiles.sh  
make

cd ../ros  
catkin build
source devel/setup.bash  
git clone git@github.com:YoshikiMaekawa2000/airsim_controller_ros.git  
```
